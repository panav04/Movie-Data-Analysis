# 🎬 Movie Data Analysis Project

## 📌 Overview  
This project is based on analyzing a movie dataset using Python. The main aim was to understand different aspects like profit, ROI, genres, and the role of actors and directors in a movie’s success.

## 🎯 Tasks Performed  
- Performed data cleaning and preprocessing  
- Calculated profit and ROI for each movie  
- Found the movie with the highest profit  
- Identified which language has the highest ROI  
- Explored different movie genres  
- Analyzed producers and actors  
- Studied the relationship between directors and actors  

## ⚙️ Tools Used  
- Python  
- Pandas  
- VS Code  

## 📊 Conclusion  
Through this project, I learned how to work with real-world data and gained insights into movie profitability, popular genres, and key contributors in the film industry.
